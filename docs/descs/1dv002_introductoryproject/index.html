1DV002 - Introductory project (7.5 hec)

Prerequisites

1DV001.

Learnings outcomes

After completing the course the student is expected to:

1. Knowledge and understanding

  1. Describe how a problem can be addresses by a software system
  2. Describe how requirements should be elicited, specified, and tested.

2. Skills and abilities

  1. Be able to express requirements in a formal way
  2. Be able to analyze a problem and design a system that addresses it
  3. Be able to implement a basic design in a programming language
  4. Be able to test a system and make sure it implements the requirements

3. Judgement and approach

  1. Take individual responsibility within a team
  2. Communicate and interact professionally within the team

Course contents

  • Introduction to projects and team work
  • Introduction to requirement elicitation
  • Introduction to software design
  • A development project
